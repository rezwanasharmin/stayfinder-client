# StayFinder - Frontend Client

StayFinder is a premium luxury vacation rental platform. This folder contains the frontend client codebase built with Next.js 16 and Tailwind CSS v4.

---

## 🚀 Tech Stack
- **Core Framework**: React 19 via Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4 (Sleek dark-mode theme)
- **Icons**: Lucide React
- **Charts**: Recharts (Analytics and metrics visualization)
- **State Management**: React Context API (Auth and Notifications)

---

## 🛠️ Installation & Run

1. Navigate to the client directory:
   ```bash
   cd client
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
